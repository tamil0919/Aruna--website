# Aruna Online Service Center

This is a responsive website created for Aruna Online Service Center.  
It provides information about Aadhaar services, Patta Chitta, and CCTV installation.  
The website is built using HTML, CSS, Bootstrap, and basic JavaScript.  
It includes a navigation bar, service section, and contact form.  
Hover effects and shadow styling are used for a modern look.  
The design works well on mobile, tablet, and desktop devices.
